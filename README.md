🌍 TripAdvisor-Style Travel Website

Welcome to my TripAdvisor-Style Travel Website Clone!
This is a responsive, beginner-friendly project built with HTML5 and CSS3.
It replicates the look and feel of TripAdvisor, showcasing modern design, travel inspiration, and a user-friendly layout.

📑 Pages & Features

Home Page: Clean navigation, search bar, and banner section.

Explore Section: Displays destinations, activities, and recommendations.

Hotels & Restaurants Section: TripAdvisor-style cards with images and details.

Footer Section: Contact info, social media links, and additional resources.

Responsive Design: Fully compatible with desktop, tablet, and mobile devices.
